FlowCSS Documentation

A modern CSS framework with 40+ animations, components, glassmorphism, 3D, and dark mode — zero dependencies, zero build step.

Installation

npm

npm install flowcss
import 'flowcss/dist/flow.css';
import 'flowcss/dist/flow.js'; // optional JS features

CDN

<link rel="stylesheet" href="https://unpkg.com/flowcss/dist/flow.css">
<script src="https://unpkg.com/flowcss/dist/flow.js" defer></script>

Tip: No PostCSS, no build step, no config. Works in any HTML, React, Vue, or Svelte project.

Basic Usage

All classes use the flow- prefix — no conflicts with other frameworks.

<div class="flow-card flow-animate-slide-up">
  <h2 class="flow-text-gradient">Hello FlowCSS</h2>
  <p class="flow-text-muted">No build step needed.</p>
  <button class="flow-btn flow-btn--primary flow-btn--ripple">Click</button>
</div>

Theming

Every design token is a CSS variable. Override globally or locally.

:root {
  --flow-primary: #7c3aed;
  --flow-font-sans: 'Poppins', sans-serif;
  --flow-radius-md: 0.25rem;
}
.my-section { --flow-primary: #ef4444; }

Dark Mode

<!-- Set on html element -->
<html data-theme="dark">

<!-- Auto toggle button (persisted via localStorage) -->
<button data-flow-theme-toggle>Toggle theme</button>

Entrance Animations

Play once on load. Chain with flow-delay-* for staggered sequences.

Class Effect
flow-animate-fade-in Fade from transparent
flow-animate-slide-up Rise from below + fade
flow-animate-slide-down Drop from above + fade
flow-animate-slide-left Slide in from right
flow-animate-slide-right Slide in from left
flow-animate-scale-in Scale 90%→100% + fade
flow-animate-pop Spring scale with overshoot
flow-animate-blur-in Unblur + fade in
flow-animate-rise Dramatic rise + scale
flow-animate-swoosh-left Slide with skew from right
flow-animate-swoosh-right Slide with skew from left
flow-animate-roll-in Roll + rotate in
flow-animate-spotlight Scale + rotate entrance
flow-animate-flip-x Flip from X axis
flow-animate-flip-y Flip from Y axis 
<h1 class="flow-animate-slide-up">Headline</h1>
<p class="flow-animate-fade-in flow-delay-200">Subtitle</p>
<button class="flow-animate-scale-in flow-delay-400">CTA</button>

Why FlowCSS

Built for what Tailwind
left behind

🎬

40+ Animations

Entrance, looping, attention-seekers, morphing, glitch, typewriter, shimmer, and scroll-triggered reveals. All via single classes.

🪟

Glassmorphism Built-in

Use flow-glass or flow-glass--dark for instant frosted-glass effects with proper backdrop-filter support.

📦

Real Components

Cards, navbars, badges, alerts, accordions, tabs, toasts, progress bars, timelines, and more — all styled and ready to use.

🎨

CSS Variables Theming

Every value — color, spacing, radius, shadow, font — is a CSS variable. Override any token globally or locally without config files.

🌑

Dark Mode

Add data-theme="dark" to the HTML element — no extra classes needed. Flow.js toggles and persists it automatically.

📐

Auto-responsive Grid

flow-auto-grid uses CSS auto-fit to be responsive with no breakpoints. Zero media-query clutter.

🧊

3D Transforms

Perspective, rotateX/Y, preserve-3d, and the flow-card-3d hover effect. First-class 3D CSS with no plugin needed.

⚡

Zero Dependencies

Just one CSS file and one optional JS file. No PostCSS, no build step, no configuration. Drop a link tag and go.
